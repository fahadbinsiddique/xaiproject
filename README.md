# Xai – Intelligence Workspace

## Overview

Narrative-driven interactive product UI demonstrating:
Raw Data → Structured Intelligence → Actionable Insight → Automation.

## Tech Stack

Next.js (App Router)
Tailwind CSS
Framer Motion
GSAP + ScrollTrigger
Three.js / React Three Fiber

## Sections

Hero – Particle to Grid transformation
Insight Flow – Scroll-based staged interaction
Dashboard Preview – Product UI mock
Signature Interaction – 3D cluster morph

## Animation Architecture

Framer Motion → UI transitions
GSAP → Scroll orchestration
R3F → 3D particle & cluster systems

## Performance Considerations

Instanced meshes
Buffer geometry
Minimal re-renders

## Run Locally

npm install
npm run dev
